  • README

    Use this form to schedule the analysis of particular SRA projects. Typically, these start with "SRP", "ERP" or "DRP". So long as pysradb can find the runs belonging to this project, this will work. Users are limited to free 5 datasets (SRPs) per year. Premium members can have 200 datasets per year, which costs US$100. email mark.ziemann{at}gmail.com to apply. All funds are invested into hardware and staff to maintain the resource. A combination code will be sent to the email address to confirm that the request originated with that person. Instructions for confirmation are given by email. Requests may take a while depending on how many datasets and how large they are. Please ensure that the email address is correct and check the spam folder before contacting admins for help. To prevent the alert going in your spam folder add www-data@dee2.io and ubuntu@dee2.io to your safe list. Completed requests data will be stored for one month only. If the server isn't that busy, it will take ~1 hr per dataset of standard size (30M reads).
