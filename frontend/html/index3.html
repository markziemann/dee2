Digital Expression Explorer 2

A repository of uniformly processed RNA-seq data

Search data

DEE2 hosts data from >40k projects involving more than a million samples. It includes widely used model organisms. If you're after data from only one or a few studies, this is the tool to try.

Data resources

  • Hosted data

    Statistics about the dataset.

  • Bulk data access

    Processed transcriptome data and metadata for your data mining pleasure.

  • Data-on-demand request service

    Can't find a dataset in DEE2? Never fear, we have set up a system to fulfil your requests.

  • Browse the catalog of bundles

    Each zip file represents an SRA Project.

  • Legacy DEE2 search

    The old search tool based on individual runs. This is only useful if you want a few samples out of a larger study. It is also much slower than the main search tool.
